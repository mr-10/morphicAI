# Morphic Ai 🤖

Morphic Ai is a high-performance, immersive, and adaptive Android application that serves as a premium wrapper for the Morphic AI chat platform (https://chat.morphic.sh). 

Built with the latest Android technologies, Morphic Ai focuses on providing a native-like experience with persistent chat history, responsive layouts for all device form factors, and a vibrant Material 3 design.

## ✨ Features

*   **🚀 Optimized WebView**: Fine-tuned for performance with hardware acceleration, high render priority, and persistent session management (DOM storage & database enabled).
*   **📱 Adaptive Layout**: Seamlessly transitions between single-pane (mobile) and list-detail (tablet/foldable) views using the Compose Material Adaptive library.
*   **📜 Persistent Chat History**: Automatically saves and organizes your chat sessions using Room database for offline access and quick resumption.
*   **🎨 Vibrant Material 3 Design**: Features a high-energy, futuristic color scheme (Electric Blue & Deep Purple) with full support for Dynamic Color and Dark Mode.
*   **🌊 Edge-to-Edge Experience**: A truly immersive, full-screen UI that respects system insets while maximizing content area.
*   **✨ Professional Loading States**: Custom shimmer skeleton screens provide smooth transitions while web content initializes.
*   **🧭 Modern Navigation**: Powered by Jetpack Navigation 3 for robust, state-driven app flow.

## 🛠️ Tech Stack

*   **UI Framework**: [Jetpack Compose](https://developer.android.com/jetpack/compose) (100%)
*   **Navigation**: [Jetpack Navigation 3](https://developer.android.com/guide/navigation/navigation-3)
*   **Architecture**: MVVM with State-driven UI
*   **Database**: [Room](https://developer.android.com/training/data-storage/room)
*   **Dependency Injection**: ViewModel with Navigation 3 Scoping
*   **Adaptive UI**: [Compose Material 3 Adaptive](https://developer.android.com/jetpack/compose/adaptive)
*   **Networking**: Retrofit & OkHttp (Infrastructure ready)
*   **Language**: Kotlin

## 🚀 Getting Started

### Prerequisites
*   Android Studio Ladybug (2024.2.1) or newer
*   JDK 17 or higher
*   Android SDK 34+

### Build Instructions
1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/morphic-ai.git
    ```
2.  Open the project in Android Studio.
3.  Wait for Gradle sync to complete.
4.  Run the app on your emulator or physical device.
